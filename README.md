# プログラミングチャット
## 制作アプリの説明
- プログラミングについて相談するSlack風チャットサービス

## 工夫した点・こだわった点
- まずはSlackに近づけるようにデザイン/UIを意識
- 実際のチャットシステムに近づけるようリッチエディタQuillを導入
- Google認証によるアカウント情報取得
- スクロールをLSに保存して続きから再開
- スレッドごとにrefを分けてメッセージを取得/管理
- いいね機能

## 次回トライしたいこと
- メッセージ編集機能
- アカウント情報をもとにしたプライベートスレッド作成機能
